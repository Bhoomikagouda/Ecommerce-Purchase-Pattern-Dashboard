# E-commerce Purchase Pattern Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes customer purchasing behavior and e-commerce sales trends. The dashboard provides insights into revenue, customer engagement, payment methods, product categories, and order distribution.

It helps businesses understand customer behavior and improve decision-making using data visualization.

---

## 🎯 Objectives
- Analyze total customers and orders
- Track total revenue and average order value
- Study customer engagement and spending patterns
- Analyze payment methods and order sizes
- Identify top-performing product categories
- Monitor daily revenue trends

---

## 📊 Dashboard Features
- Total Customers KPI
- Total Orders KPI
- Total Revenue KPI
- Average Order Value
- Daily Revenue Trend Analysis
- Revenue by Category
- Top Product Categories
- Customer Engagement vs Spending
- Payment Method Distribution
- Order Size Distribution
- Interactive Filters and Navigation

---

## 🛠️ Tools & Technologies Used
- Power BI
- Excel Dataset
- DAX Functions
- Data Cleaning & Transformation
- Data Visualization

---

## 📈 Key Insights
- Total revenue reached approximately $5.08M.
- Average order value is around $0.20K.
- Credit card payments are more common than PayPal.
- Low-sized orders contribute the majority of purchases.
- Customer engagement impacts spending behavior.
